# 🗣️ Talk Note Demo

A simple iOS app that lets users record voice notes and save them with timestamps.

## 🚀 Features

- 🎙 Record voice notes
- 📝 Store notes with timestamp
- 📂 List and manage recorded notes
- 🧼 Clean UI with SwiftUI

## 🛠 Tech Stack

- Swift
- SwiftUI
- AVFoundation
